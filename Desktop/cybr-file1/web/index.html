INSTALLATION PROTOCOL

Deploy Cyber-File in your environment

Method 1: Python Source

TERMINAL bash

$ git clone https://github.com/yourusername/cyber-file.git

$ cd cyber-file

$ python -m venv venv

$ venv\Scripts\activate # Windows

$ pip install -r requirements.txt

$ python cyberfile_v2.py

Method 2: Build EXE

TERMINAL powershell

$ pip install pyinstaller

$ pyinstaller --onefile --console --icon=images/hacking.ico cyberfile_v2.py

# Output in dist/ directory

$ ./dist/cyberfile_v2.exe
