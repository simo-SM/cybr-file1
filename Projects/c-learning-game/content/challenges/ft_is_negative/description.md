ft_is_negative
--------------
Task: Implement a function that prints N when n is negative, otherwise P. Always print a newline at the end.

Prototype:
  void ft_is_negative(int n);

Examples:
- ft_is_negative(-5) prints N then newline
- ft_is_negative(0) prints P then newline
- ft_is_negative(42) prints P then newline
