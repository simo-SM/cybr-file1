ft_print_comb
-------------
Task: Print all unique combinations of three different digits in ascending order.
Format:
- Digits are from 0 to 9, strictly increasing by position.
- Each combination is printed as three digits.
- Combinations are separated by a comma followed by a space.
- After the final combination, print a newline.

Example start:
012, 013, 014, 015, ...

Example end:
..., 789
